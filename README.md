# Demo

Приложение для демонстрации работы с базой данных, авторизацией и управлением пользователями.

## Запуск приложения

```bash
python run.py
```

## Генерация документации

Для генерации документации используется [pdoc](https://pdoc.dev/).

### Установка pdoc

pdoc уже включён в `requirements.txt`. Установить все зависимости можно командой:

```bash
pip install -r requirements.txt
```

### Создание документации

```bash
python -m pdoc Demo --output-dir docs
```

Эта команда анализирует docstring'и в модулях пакета `Demo` и генерирует HTML-документацию в папку `docs/`.

Результат:
- `docs/index.html` — точка входа (перенаправляет на `Demo.html`)
- `docs/Demo.html` — главная страница документации пакета `Demo`
- `docs/Demo/` — детальная документация по подмодулям: `Connection`, `Controllers`, `Models`, `Views`

После генерации откройте `docs/index.html` в браузере для просмотра.
